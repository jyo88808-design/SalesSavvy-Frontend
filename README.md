# 🛒 SalesSavvy Frontend

## 📌 About

SalesSavvy Frontend is a responsive e-commerce web application developed using React.js. It provides a modern user interface for browsing products, managing the shopping cart, and interacting with the backend through REST APIs.

## 🚀 Features

- 👤 User Registration & Login
- 🛍️ Browse Products
- 🔍 Product Categories
- 🛒 Add to Cart
- ❤️ User-Friendly Interface
- 📱 Responsive Design

## 🛠️ Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3
- React Router
- Axios
- Git
- GitHub

## ▶️ Installation

```bash
git clone https://github.com/jyo88808-design/SalesSavvy-Frontend.git
cd SalesSavvy-Frontend
npm install
npm run dev
```

Open your browser:

```text
http://localhost:5173/
```
